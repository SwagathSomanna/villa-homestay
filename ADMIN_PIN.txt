ADMIN CONTROL PANEL ACCESS INFORMATION
===========================================

Admin PIN: 7845

This PIN is required to access the admin control panel at:
http://localhost:4000/admin.html

SECURITY NOTES:
- The PIN is hashed using SHA-256 for secure storage
- Admin sessions expire after 1 hour of inactivity
- The PIN can be changed by setting the ADMIN_PIN environment variable
- For production, use a strong, unique PIN and keep it secure

The admin panel allows you to:
- Update room and floor pricing
- Block dates for rooms/floors/villa
- Update gallery images
- View and confirm bookings
- Receive email notifications when guests mark payments as paid

When guests mark a payment as paid, the admin receives:
- Guest Name
- Email ID
- Phone Number
- Booking Type
- Check-In Date
- Check-Out Date
- Number of nights
- Number of adults and children
